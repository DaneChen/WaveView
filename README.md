###How To Use
add a WaveView into your XML.

```
<com.wingsofts.waveview.WaveView
      wave:mode="triangle"
      android:color="#6337B5"
      wave:waveWidth="10"
      wave:waveCount="20"
      android:layout_width="wrap_content"
      android:layout_height="wrap_content" />
```
 Enjoy
